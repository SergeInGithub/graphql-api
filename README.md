# 💠 graphql-api

## ℹ️ Description
This is A full-stack app built to explore and demonstrate some advantages of building a GraphQL API 

⚠️ This project is currently still in progress

## 🔮 Tech Stack

- Tailwind CSS
- React
- Typescript
- Node.js
- Express
- TypeORM
- PostgreSQL
- PostGraphile
- Apollo GraphQL Client
- GraphQL Codegen
- Postman

## 📦 Installation

```bash
pnpm install
```

## 🚀 Running the app

```bash
# development
pnpm start
```
